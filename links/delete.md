# Delete Link between Items

## Request
* **URL**: `/auth/v1/links/:linkId`
* **Method**: `DELETE`
* **Headers**:
    * `Authorization: ApiKey YOUR_API_KEY`
    * `workspaceId: YOUR_WORKSPACE_ID`
* **Request Parameters**:
    * `linkId`: Link ID
* **Query Parameters**: None
* **Request Body**: None
## Response
* **Status**: `200 OK`
* **Response Body**:
None
