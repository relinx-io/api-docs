* **URL**: `/auth/v1/values/:valueId`
* **Method**: `DELETE`
* **Headers**:
    * `Authorization: ApiKey YOUR_API_KEY`
    * `workspaceId: YOUR_WORKSPACE_ID`
* **Request Parameters**:
    * `valueId`: Value ID
* **Query Parameters**: None
* **Request Body**: None

## Response
* **Status**: `200 OK`
* **Response Body**:
None
