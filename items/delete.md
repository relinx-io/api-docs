# Delete Item

## Request
* **URL**: `/auth/v1/items/:itemId`
* **Method**: `DELETE`
* **Headers**:
    * `Authorization: ApiKey YOUR_API_KEY`
    * `workspaceId: YOUR_WORKSPACE_ID`
* **Request Parameters**:
    * `itemId`: Item ID
* **Query Parameters**: None
* **Request Body**: None

## Response
* **Status**: `200 OK`
* **Response Body**: None
